📋 ЧТО ДЕЛАТЬ НА СТРАНИЦЕ RAILWAY:

1️⃣ Нажмите большую кнопку:
   
   🔵 "Deploy from GitHub repo"

───────────────────────────────────────────────────────────────

2️⃣ Если первый раз:
   
   Появится окно "Configure GitHub App"
   
   ✅ Нажмите "Install & Authorize"
   ✅ Выберите "Only select repositories"
   ✅ Найдите и выберите: nano-banana
   ✅ Нажмите "Install"

───────────────────────────────────────────────────────────────

3️⃣ Выберите репозиторий:
   
   📂 alexpumpforyou-art/nano-banana
   
   ✅ Кликните на него

───────────────────────────────────────────────────────────────

4️⃣ Railway автоматически начнёт деплой!
   
   Вы увидите:
   - 🔵 Синий статус "Building..."
   - Логи в реальном времени
   - Через 2-3 минуты статус станет 🟢 зелёным

📋 ШАГ 5: ПОЛУЧИТЬ ПУБЛИЧНЫЙ URL

1️⃣ Перейдите на вкладку "Settings" (вверху)

2️⃣ Найдите раздел "Networking" или "Domains"

3️⃣ Нажмите "Generate Domain"

4️⃣ Скопируйте URL (например: nano-banana-production.up.railway.app)
