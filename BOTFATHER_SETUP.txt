───────────────────────────────────────────────────────────────
ШАГ 1: Откройте @BotFather в Telegram
───────────────────────────────────────────────────────────────

Ссылка: https://t.me/BotFather

───────────────────────────────────────────────────────────────
ШАГ 2: Установите команды
───────────────────────────────────────────────────────────────

1. Отправьте команду: /setcommands

2. Выберите вашего бота

3. Скопируйте и вставьте ВСЁ это:

start - Начать работу с ботом
balance - Мой баланс и статистика
buy - Купить кредиты
referral - Реферальная программа
help - Помощь и инструкции
paysupport - Вопросы по оплате (обязательно для Stars)
support - Общая поддержка (обязательно для Stars)
terms - Условия использования (обязательно для Stars)

4. Нажмите ОТПРАВИТЬ

───────────────────────────────────────────────────────────────
ШАГ 3: Заполните информацию о боте
───────────────────────────────────────────────────────────────

1. /setdescription
   Выберите бота
   Напишите:
   
   Nano Banana - AI-бот для генерации и редактирования изображений 
   с помощью Google Gemini AI. Создавайте уникальные изображения по 
   текстовому описанию или редактируйте существующие фотографии. 
   Простой, быстрый и качественный результат!

2. /setabouttext
   Выберите бота
   Напишите:
   
   🎨 Генерация изображений
   ✏️ Редактирование фото
   💎 Кредиты за Stars
   👥 Реферальная программа

3. /setuserpic
   Выберите бота
   Загрузите красивую картинку (квадратную!)

───────────────────────────────────────────────────────────────
ШАГ 4: Включите Telegram Stars
───────────────────────────────────────────────────────────────

1. @BotFather → /mybots

2. Выберите вашего бота

3. Bot Settings

4. Payments

5. Выберите "Telegram Stars" или "Use Telegram Stars"

6. Подтвердите условия

───────────────────────────────────────────────────────────────
ШАГ 5: Дождитесь одобрения
───────────────────────────────────────────────────────────────

⏰ После настройки может потребоваться 24-48 часов для активации
   Telegram Stars. Telegram проверяет боты перед одобрением.

───────────────────────────────────────────────────────────────
ШАГ 6: Проверьте что всё работает
───────────────────────────────────────────────────────────────

1. Дождитесь деплоя на Railway (1-2 минуты)

2. Откройте вашего бота в Telegram

3. Попробуйте команды:
   /start
   /paysupport
   /support
   /terms

4. Попробуйте купить кредиты:
   /buy → Выберите пакет

5. Если форма оплаты открывается - ОТЛИЧНО! Stars работают!

6. Если висит загрузка - смотрите логи Railway:
   Railway → Deployments → View Logs
   
   Ищите строку: 🔔 PRE_CHECKOUT_QUERY ПОЛУЧЕН!
   
   Если НЕТ этой строки - Stars еще не активированы в BotFather
